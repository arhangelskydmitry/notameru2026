# ✅ Ограничение высоты слайдера и центрирование изображений - Выполнено!

## 🎯 Что было сделано

### 1. ✅ Установлено ограничение максимальной высоты

**Добавлено:**
```css
.main-slider {
    height: 100%;
    max-height: 500px;  /* Ограничиваем максимальную высоту */
}
```

**Результат:**
- Слайдер не будет выше 500px
- Нижняя граница совпадает с правым блоком
- Высота адаптируется, но не превышает лимит

---

### 2. ✅ Изображение центрируется по геометрическому центру

**Добавлено:**
```css
.slider-item img {
    object-fit: cover;
    object-position: center center;  /* Центрируем изображение */
}
```

**Результат:**
- Геометрический центр изображения совпадает с центром слайдера
- При подрезке обрезаются края равномерно сверху и снизу
- Главный объект на фото всегда в центре

---

## 📊 Как это работает

### Object-position: center center

**Принцип работы:**
```
Оригинальное изображение (большое):
┌─────────────────────┐
│         ▲           │ ← Верх (обрезается)
│         │           │
│    ┌────┼────┐      │
│    │    ●    │      │ ← Центр (сохраняется)
│    └────┼────┘      │
│         │           │
│         ▼           │ ← Низ (обрезается)
└─────────────────────┘

В слайдере (500px max-height):
    ┌─────────┐
    │    ●    │ ← Центр изображения = центр слайдера
    └─────────┘
```

### Max-height: 500px

**Логика:**
```css
height: 100%;        /* Пытается заполнить всю высоту родителя */
max-height: 500px;   /* Но не больше 500px */
```

**Результат:**
- Если правый блок < 500px → слайдер = высота правого блока
- Если правый блок > 500px → слайдер = 500px (лимит)

## 🎨 Технические детали

### CSS свойства:

| Свойство | Значение | Описание |
|----------|----------|----------|
| `height` | `100%` | Базовая высота |
| `max-height` | `500px` | Максимальная высота |
| `object-fit` | `cover` | Заполняет область |
| `object-position` | `center center` | Центрирование |

### Object-position синтаксис:

```css
/* Центр по обеим осям */
object-position: center center;

/* То же самое короткая запись */
object-position: center;

/* Другие варианты: */
object-position: top;        /* Верх по центру */
object-position: bottom;     /* Низ по центру */
object-position: left;       /* Лево по центру */
object-position: 50% 30%;    /* Кастомная позиция */
```

### Как работает центрирование:

```
Большое изображение 1200x800px
Слайдер 800x500px

По горизонтали:
1200px изображение → обрезаем по 200px слева и справа
Центр 600px остаётся

По вертикали:
800px изображение → обрезаем по 150px сверху и снизу
Центр 500px остаётся

Результат: Центральная часть 800x500px
```
