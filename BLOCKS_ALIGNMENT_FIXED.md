### 2. ✅ Выровнена высота блока "Последние новости"

**Было:**
```html
<div style="display: grid; grid-template-columns: 2fr 1fr; gap: 15px; margin-bottom: 40px;">
    <div>
        <div class="main-slider" style="...">
```

**Стало:**
```html
<div style="display: grid; grid-template-columns: 2fr 1fr; gap: 15px; margin-bottom: 20px; align-items: start;">
    <div style="height: 100%;">
        <div class="main-slider" style="height: calc(100% - 48px);">
```

**Изменения:**
1. Добавлен `align-items: start` к grid-контейнеру
2. Добавлен `height: 100%` к обёртке слайдера
3. Добавлен `height: calc(100% - 48px)` к слайдеру (вычитаем высоту заголовка 48px)

**Результат:**
- Слайдер растягивается по высоте правого блока
- Высота "Последние новости" = высота "Интервью" + gap + "Релизы"
- Блоки выровнены по верхнему краю

## 🎨 Технические детали

### Grid Layout:
```css
display: grid;
grid-template-columns: 2fr 1fr;  /* Слева в 2 раза шире */
gap: 15px;                       /* Между колонками */
margin-bottom: 20px;             /* До следующего блока */
align-items: start;              /* Выравнивание по верху */
```

### Высота слайдера:
```css
/* Обёртка */
height: 100%;

/* Слайдер */
height: calc(100% - 48px);  /* Вычитаем высоту красного заголовка */
```

**Расчёт:**
- Заголовок: 48px (padding 12px × 2 + font-size)
- Слайдер: 100% - 48px = автоматическая подстройка под правый блок
