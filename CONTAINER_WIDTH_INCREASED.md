# ✅ Увеличение ширины контейнеров - Выполнено!

## 📊 Изменения размеров

### Было:
```css
.container {
    max-width: 1200px;
    margin: 0 auto;
    padding: 0 15px;
}
```

**Фактические размеры:**
- **Максимальная ширина контента:** 1200px
- **Отступы слева и справа:** по 15px (всего 30px)
- **Общая ширина контейнера:** 1200px + 30px = 1230px
- **На экране 1920px:** отступы слева и справа по ~345px ((1920 - 1230) / 2)

### Стало:
```css
.container {
    max-width: 1400px;  /* +200px */
    margin: 0 auto;
    padding: 0 15px;
}
```

**Фактические размеры:**
- **Максимальная ширина контента:** 1400px
- **Отступы слева и справа:** по 15px (всего 30px)
- **Общая ширина контейнера:** 1400px + 30px = 1430px
- **На экране 1920px:** отступы слева и справа по ~245px ((1920 - 1430) / 2)

## 📋 Что это затрагивает

Изменение `.container` влияет на все элементы внутри контейнера:

### ✅ Шапка сайта
- Логотип и баннер
- Место для меню и поиска стало больше

### ✅ Основной контент
- Слайдер новостей
- Сетка новостей
- Боковая панель (интервью, релизы)

### ✅ Навигация
- Пункты меню
- Поиск

### ❌ НЕ затрагивает
- Бегущая строка (она на 100% ширины экрана)
- Красная навигационная панель (она тоже на 100%)

1. **Больше места для контента** (+200px в ширину)
2. **Меньше пустого пространства** по бокам
3. **Лучшее использование экрана** на широких мониторах
4. **Более современный вид** - шире контент
5. **Больше места для меню** и поиска в навигации
